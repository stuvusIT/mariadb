# Role Name

This ansible role configures an mariadb server.


## Requirements

An apt based packagmanager and systemd


## Role Variables

Read the mariadb doc to get to know what vars can be set


## Example Playbook


### Playbook

```yml
hosts: wordpress
roles:
    - mariadb
```


### Result

Sets up a running mariadb instance
